English

This project aims to help you in preparations for exams by providing means in training single subjects in mind map format. It targets primarily students of universities, but you can use it for improving grades in other schools, as well. For training you need to sum up your school materials in form of mind maps: collect the points that you need to know and to remeber in a list, and let the program train you. The software will present the subjects, while you yourself decide, if you could recalled the points correctly. The usual training is done in text form, but you can also use pictures. If you have a picture that needs to be described for correct answer, then place a reference to it in first subpoint.


If you have one picture as assignment and another one as answer then you can place a reference to answer picture in second subpoint.


Following references to pictures are supported:

  1. A full path to the picture.
  2. A path, relative to pictures folder.
  3. A path, relative to documents folder.

If you use two pictures then the second picture must have same dimensions as first.

I wish you all the best grades and the brightes career prospects.
